# 行情接入实现方案

本文档提供**行情接入**功能的详细实现方案，包括：**读取解析输入的行情信息** 和 **撮合时考虑行情信息**（保证买价、卖价和对手方价格保持一致）。

> **当前实现说明**：系统已实现行情接入。实际实现与下文方案略有差异：**行情由盘口自动计算**——`OpenOrderStore` 在挂单增/删时按买卖档计算买一/卖一并写入 `MarketDataStore`；查询接口为 `GET /api/market-data`；下单前校验使用 `MarketPriceValidator`（EXACT/INSIDE 模式）。可选 `POST /internal/market-data/snapshot` 批量写入外部快照。详见 `docs/core模块功能与接口说明.md` 第 3.5 节。

---

## 一、方案概述

### 1.1 架构设计

```
外部行情源/手动输入
    ↓
[行情接收接口] POST /api/market-data
    ↓
[行情存储服务] MarketDataStore (内存 ConcurrentHashMap)
    ↓
[行情校验器] MarketDataValidator (下单前校验)
    ↓
[订单服务] OrderService.placeOrder (集成行情校验)
```

### 1.2 实现位置

- **模块**：`trading-core`（不新建模块，复用现有结构）
- **存储**：内存存储（`MarketDataStore`），可选持久化到数据库
- **接口**：`MarketDataController`（接收行情 JSON Array）
- **校验**：`MarketDataValidator`（类似 `OrderValidation`）
- **集成**：在 `OrderService.placeOrder` 中，对敲风控之后、引擎下单之前增加行情校验

### 2.2 行情存储服务（新增）

**文件**：`trading-core/src/main/java/com/trading/core/store/MarketDataStore.java`

**实现方式**：
- 使用 `ConcurrentHashMap<String, MarketDataRecord>`，key = `market + ":" + securityId`（如 "XSHG:600030"）
- 线程安全，支持并发读写
- 可选：增加过期机制（如行情超过 5 分钟未更新则视为失效）

**代码结构**：

```java
package com.trading.core.store;

import com.trading.common.model.request.MarketData;
import org.springframework.stereotype.Component;

import java.time.LocalDateTime;
import java.util.concurrent.ConcurrentHashMap;

/**
 * 行情数据存储（内存）
 * Key: market + ":" + securityId (如 "XSHG:600030")
 */
@Component
public class MarketDataStore {

    private static class MarketDataRecord {
        MarketData data;
        LocalDateTime updatedAt;
        
        MarketDataRecord(MarketData data) {
            this.data = data;
            this.updatedAt = LocalDateTime.now();
        }
    }

    private final ConcurrentHashMap<String, MarketDataRecord> store = new ConcurrentHashMap<>();
    
    /** 行情有效期（秒），超过此时间视为失效 */
    private static final long VALIDITY_SECONDS = 300; // 5分钟

    /**
     * 更新行情
     */
    public void update(MarketData data) {
        if (data == null || data.getMarket() == null || data.getSecurityId() == null) {
            return;
        }
        String key = buildKey(data.getMarket(), data.getSecurityId());
        store.put(key, new MarketDataRecord(data));
    }

    /**
     * 批量更新行情
     */
    public void updateBatch(java.util.List<MarketData> dataList) {
        if (dataList == null) return;
        for (MarketData data : dataList) {
            update(data);
        }
    }

    /**
     * 获取行情
     */
    public MarketData get(String market, String securityId) {
        String key = buildKey(market, securityId);
        MarketDataRecord record = store.get(key);
        if (record == null) {
            return null;
        }
        // 检查是否过期
        if (isExpired(record)) {
            store.remove(key);
            return null;
        }
        return record.data;
    }

    /**
     * 检查行情是否存在且有效
     */
    public boolean hasValidMarketData(String market, String securityId) {
        MarketData data = get(market, securityId);
        return data != null && data.getBidPrice() != null && data.getAskPrice() != null;
    }

    private String buildKey(String market, String securityId) {
        return market + ":" + securityId;
    }

    private boolean isExpired(MarketDataRecord record) {
        return record.updatedAt.plusSeconds(VALIDITY_SECONDS).isBefore(LocalDateTime.now());
    }

    /**
     * 清空所有行情（用于测试或重置）
     */
    public void clear() {
        store.clear();
    }

    /**
     * 获取所有行情数量（用于监控）
     */
    public int size() {
        return store.size();
    }
}
```

### 2.3 行情校验器（新增）

**文件**：`trading-core/src/main/java/com/trading/core/risk/MarketDataValidator.java`

**实现方式**：
- 类似 `OrderValidation`，静态方法 `validateAgainstMarketData`
- 校验逻辑：
  - 买单（B）：订单价格 ≤ askPrice（不能高于卖一价）
  - 卖单（S）：订单价格 ≥ bidPrice（不能低于买一价）
- 若行情不存在，可配置为“允许通过”或“拒绝”（建议配置项控制）

**代码结构**：

```java
package com.trading.core.risk;

import com.trading.common.enums.RejectCode;
import com.trading.common.model.request.MarketData;
import com.trading.common.model.request.OrderRequest;
import com.trading.common.model.response.OrderRejectResponse;
import com.trading.core.store.MarketDataStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Component;

/**
 * 行情校验：下单时校验订单价格是否在行情范围内
 * 需求：撮合时需要保证买价、卖价和对手方价格保持一致
 */
@Component
public class MarketDataValidator {

    private final MarketDataStore marketDataStore;
    
    /** 是否启用行情校验（配置项） */
    private boolean enabled = true;
    
    /** 行情不存在时的行为：true=拒绝，false=允许通过 */
    private boolean rejectIfNoMarketData = false;

    @Autowired
    public MarketDataValidator(MarketDataStore marketDataStore) {
        this.marketDataStore = marketDataStore;
    }

    /**
     * 校验订单价格是否在行情范围内
     * @param req 订单请求
     * @return 若违反行情规则则返回 OrderRejectResponse，否则返回 null
     */
    public OrderRejectResponse validateAgainstMarketData(OrderRequest req) {
        if (!enabled) {
            return null; // 未启用，直接通过
        }

        if (req == null || req.getMarket() == null || req.getSecurityId() == null) {
            return null; // 基本信息缺失，由 OrderValidation 处理
        }

        MarketData marketData = marketDataStore.get(req.getMarket(), req.getSecurityId());
        
        // 行情不存在
        if (marketData == null || marketData.getBidPrice() == null || marketData.getAskPrice() == null) {
            if (rejectIfNoMarketData) {
                return reject(req, RejectCode.INVALID_MARKET_DATA, 
                    "行情数据不存在: " + req.getMarket() + ":" + req.getSecurityId());
            }
            return null; // 允许通过（行情可能未推送）
        }

        Double orderPrice = req.getPrice();
        if (orderPrice == null) {
            return null; // 价格为空由 OrderValidation 处理
        }

        String side = req.getSide();
        Double bidPrice = marketData.getBidPrice();
        Double askPrice = marketData.getAskPrice();

        // 买单：订单价格不能高于卖一价（askPrice）
        if ("B".equals(side) || "BUY".equalsIgnoreCase(side)) {
            if (orderPrice > askPrice) {
                return reject(req, RejectCode.PRICE_OUT_OF_MARKET_RANGE,
                    String.format("买单价格 %.2f 高于卖一价 %.2f", orderPrice, askPrice));
            }
        }
        // 卖单：订单价格不能低于买一价（bidPrice）
        else if ("S".equals(side) || "SELL".equalsIgnoreCase(side)) {
            if (orderPrice < bidPrice) {
                return reject(req, RejectCode.PRICE_OUT_OF_MARKET_RANGE,
                    String.format("卖单价格 %.2f 低于买一价 %.2f", orderPrice, bidPrice));
            }
        }

        return null; // 通过校验
    }

    private OrderRejectResponse reject(OrderRequest req, RejectCode code, String text) {
        OrderRejectResponse r = new OrderRejectResponse();
        if (req != null) {
            r.setClOrderId(req.getClOrderId());
            r.setMarket(req.getMarket());
            r.setSecurityId(req.getSecurityId());
            r.setSide(req.getSide());
            r.setQty(req.getQty());
            r.setPrice(req.getPrice());
            r.setShareholderId(req.getShareholderId());
        }
        r.setRejectCode(code.getCode());
        r.setRejectText(text);
        return r;
    }

    // Getter/Setter for configuration
    public boolean isEnabled() {
        return enabled;
    }

    public void setEnabled(boolean enabled) {
        this.enabled = enabled;
    }

    public boolean isRejectIfNoMarketData() {
        return rejectIfNoMarketData;
    }

    public void setRejectIfNoMarketData(boolean rejectIfNoMarketData) {
        this.rejectIfNoMarketData = rejectIfNoMarketData;
    }
}
```

### 2.4 行情接收接口（新增）

**文件**：`trading-core/src/main/java/com/trading/core/controller/MarketDataController.java`

**实现方式**：
- `POST /api/market-data`：接收 JSON Array（符合需求文档格式）
- 解析并批量更新到 `MarketDataStore`
- 返回成功/失败响应

**代码结构**：

```java
package com.trading.core.controller;

import com.trading.common.model.request.MarketData;
import com.trading.core.store.MarketDataStore;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

import java.util.HashMap;
import java.util.List;
import java.util.Map;

/**
 * 行情数据接收接口
 * 需求：能读取解析输入的行情信息
 */
@RestController
@RequestMapping("/api")
public class MarketDataController {

    private static final Logger log = LoggerFactory.getLogger(MarketDataController.class);

    private final MarketDataStore marketDataStore;

    @Autowired
    public MarketDataController(MarketDataStore marketDataStore) {
        this.marketDataStore = marketDataStore;
    }

    /**
     * 接收行情数据（JSON Array）
     * POST /api/market-data
     * Body: [{"market":"XSHG","securityId":"600030","bidPrice":10.5,"askPrice":10.6}, ...]
     */
    @PostMapping("/market-data")
    public ResponseEntity<?> receiveMarketData(@RequestBody List<MarketData> marketDataList) {
        if (marketDataList == null || marketDataList.isEmpty()) {
            return ResponseEntity.badRequest().body(createErrorResponse("行情数据为空"));
        }

        int successCount = 0;
        int failCount = 0;

        for (MarketData data : marketDataList) {
            try {
                // 基本校验
                if (data.getMarket() == null || data.getSecurityId() == null) {
                    failCount++;
                    log.warn("Invalid market data: market or securityId is null");
                    continue;
                }
                if (data.getBidPrice() == null || data.getAskPrice() == null) {
                    failCount++;
                    log.warn("Invalid market data: bidPrice or askPrice is null for {}:{}", 
                        data.getMarket(), data.getSecurityId());
                    continue;
                }
                if (data.getBidPrice() <= 0 || data.getAskPrice() <= 0) {
                    failCount++;
                    log.warn("Invalid market data: price <= 0 for {}:{}", 
                        data.getMarket(), data.getSecurityId());
                    continue;
                }
                if (data.getBidPrice() > data.getAskPrice()) {
                    failCount++;
                    log.warn("Invalid market data: bidPrice > askPrice for {}:{}", 
                        data.getMarket(), data.getSecurityId());
                    continue;
                }

                marketDataStore.update(data);
                successCount++;
                log.debug("Updated market data: {}:{} bid={} ask={}", 
                    data.getMarket(), data.getSecurityId(), data.getBidPrice(), data.getAskPrice());
            } catch (Exception e) {
                failCount++;
                log.error("Error updating market data: {}:{}", 
                    data.getMarket(), data.getSecurityId(), e);
            }
        }

        Map<String, Object> response = new HashMap<>();
        response.put("success", failCount == 0);
        response.put("total", marketDataList.size());
        response.put("successCount", successCount);
        response.put("failCount", failCount);

        if (failCount == 0) {
            return ResponseEntity.ok(response);
        } else {
            return ResponseEntity.status(HttpStatus.PARTIAL_CONTENT).body(response);
        }
    }

    /**
     * 查询行情（用于调试或前端展示）
     * GET /api/market-data?market=XSHG&securityId=600030
     */
    @GetMapping("/market-data")
    public ResponseEntity<?> getMarketData(
            @RequestParam String market,
            @RequestParam String securityId) {
        MarketData data = marketDataStore.get(market, securityId);
        if (data == null) {
            return ResponseEntity.notFound().build();
        }
        return ResponseEntity.ok(data);
    }

    /**
     * 清空所有行情（用于测试）
     * DELETE /api/market-data
     */
    @DeleteMapping("/market-data")
    public ResponseEntity<?> clearMarketData() {
        marketDataStore.clear();
        return ResponseEntity.ok().body(createSuccessResponse("所有行情已清空"));
    }

    private Map<String, Object> createErrorResponse(String message) {
        Map<String, Object> response = new HashMap<>();
        response.put("success", false);
        response.put("message", message);
        return response;
    }

    private Map<String, Object> createSuccessResponse(String message) {
        Map<String, Object> response = new HashMap<>();
        response.put("success", true);
        response.put("message", message);
        return response;
    }
}
```

## 六、注意事项

1. **行情不存在时的行为**：建议默认允许订单通过（`rejectIfNoMarketData=false`），避免因行情未推送导致无法下单。
2. **行情更新频率**：内存存储适合高频更新，若需历史查询则考虑数据库。
3. **线程安全**：`MarketDataStore` 使用 `ConcurrentHashMap`，保证并发安全。
4. **价格精度**：行情价格与订单价格均为 `Double`，注意浮点数比较精度问题（可考虑使用 `BigDecimal` 或误差范围比较）。
