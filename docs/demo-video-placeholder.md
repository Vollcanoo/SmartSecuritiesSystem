# 演示视频

将录制好的演示视频命名为 **`demo.mp4`** 放入此目录即可。

README.md 会自动通过以下路径引用并在 GitHub 页面内嵌播放：

```
docs/demo.mp4
```

## 录制建议

演示内容可按以下顺序录制：

1. 启动系统（`./start-all.sh`），打开 http://localhost:8000
2. 用户管理：新建股东账户
3. 行情页：展示实时 A 股行情自动刷新
4. 交易页：下买单 + 下卖单，触发撮合成交
5. 挂单页 & 历史页：查看订单状态
6. AI 分析页：输入 API Key，选 Qwen 模型，提问分析交易数据

推荐使用 macOS 自带的 **QuickTime 播放器 → 新建屏幕录制**，或 OBS Studio。
