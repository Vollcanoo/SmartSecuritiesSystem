系统配置

  • MARKET_CODE → XSHG（上交所）| XSHE（深交所）| BJSE（北交所）
  • ORDER_SIDE → B（买入 BID）| S（卖出 ASK）
  • RISK_CTRL → 对敲风控：同一股东号 + 同市场 + 同证券 ≠ 同时买卖
  • PRICE_CHK → 涨跌停 ±10% | 偏离度 ±10%（基于行情数据）
  • ENDPOINTS → Admin :8080 | Core :8081 | Gateway :9000 | Frontend :8000
